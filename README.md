# Mini PaaS Platform

A lightweight Platform-as-a-Service (PaaS) application built with Python that allows users to deploy and manage applications through a simple web interface.

## Features

- Application deployment
- Deployment status tracking
- Simple dashboard
- Python backend
- Lightweight architecture

## Tech Stack

- Python
- Flask
- HTML/CSS
- GitHub

## Project Structure

```
app.py
deploy.py
README.md
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/neeraj181-dot/Paas.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```
